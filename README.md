# Bitbrew Cafe

This is a project to showcase Blockonomics' API and how it can be used to generate a Bitcoin address and listen for payments.

## Backend

The backend is a simple Node.js app that uses the Blockonomics API to generate a Bitcoin address and listen for payments.

### Installation

```bash
pnpm install
```

### Usage

```bash
pnpm start
```

## Frontend

The frontend is a simple React app that uses the Blockonomics API to generate a Bitcoin address and listen for payments.

### Installation

```bash
pnpm install
```

### Usage

```bash
pnpm dev
```
